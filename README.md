# img-grid-task

## 😎 Description

React & Express.js test project.
React SPA fetches array of images from Express app.
Global tech stack: Typescript, React.js, Express.js, Jest, SCSS.

## 👨‍💻 Install & run & test

👉 Frontend app runs on 3000 port
```
cd frontend
npm install
npm start

// run unit-tests
npm run test
```
👉 Backend app runs on 4000 port
```
cd backend
npm install
npm start

// run unit-tests
npm run test
```
